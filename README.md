# 🤖 Alura Agente - TechShop

Proyecto final del programa Oracle Next Education + Alura Latam. Un agente que responde preguntas sobre la documentación de TechShop, una tienda online.

---

## 🛠️ Lo que usé

- Python
- LangChain y LangGraph
- Groq (modelo Llama 3.3)
- PyPDF (para leer el PDF)
- Google Colab

---

## ▶️ Cómo usarlo

1. Abrí el archivo `.ipynb` en Google Colab.
2. Poné tu API Key de Groq en los secrets de Colab (nombre: `GROQ_API_KEY`).
3. Ejecutá las celdas en orden.
4. Hacé una pregunta y el agente responde.

---

## ❓ Preguntas de ejemplo

- ¿Puedo devolver un producto si ya abrí el empaque?
- ¿Qué métodos de pago aceptan?
- ¿Cuánto cuesta el envío express?
